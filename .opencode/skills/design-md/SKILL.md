## What I do

- Provide ready-to-use DESIGN.md files from popular websites
- Help AI agents understand design systems for consistent UI generation
- Include design tokens for colors, typography, components, and layout

## When to use me

Use this when you want to:
- Generate UI that matches a specific website's design language
- Create consistent design systems for your project
- Get inspiration from real-world design implementations

## How to use

1. Choose a DESIGN.md from the collection in `design-systems/` directory (e.g., `design-systems/airbnb/DESIGN.md`, `design-systems/stripe/DESIGN.md`)
2. Tell your AI agent: "Build me a page that looks like this" and reference the DESIGN.md file path

## Available design systems

The collection includes 73 DESIGN.md files from companies like:
- **Tech**: Apple, Google, Meta, NVIDIA, Spotify, Uber
- **Fintech**: Stripe, Coinbase, Revolut, Wise
- **E-commerce**: Airbnb, Nike, Shopify, Starbucks
- **Automotive**: BMW, Ferrari, Tesla, Lamborghini
- **Developer tools**: Vercel, Linear, Notion, Figma

## File structure

Each DESIGN.md follows the Stitch format with sections:
1. Visual Theme & Atmosphere
2. Color Palette & Roles  
3. Typography Rules
4. Component Stylings
5. Layout Principles
6. Depth & Elevation
7. Do's and Don'ts
8. Responsive Behavior
9. Agent Prompt Guide
